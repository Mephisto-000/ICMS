# RK4 example


Solve : 
dx/dt = y
dy/dt = u(1 - x^2)y - x

u = sin(t)^(1/5)
init : 
        step time : 0.01 (s.)
        simulation time : 150 (s.)
        initial conditions : y(0) = 0.01, x(0) = 0
